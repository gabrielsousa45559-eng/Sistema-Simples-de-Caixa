# Sistema Simples de Caixa

Sistema simples de controle de caixa desenvolvido por mim utilizando **HTML, CSS e JavaScript**, com foco em controle básico de entradas e saídas.

Este projeto faz parte do meu **portfólio pessoal**, voltado para demonstrar lógica de programação, organização de código e desenvolvimento de sistemas simples para uso administrativo.

---

## Funcionalidades

- Registro de entradas e saídas de valores
- Cálculo automático do saldo
- Listagem das movimentações em tabela
- Exclusão de registros
- Interface simples, limpa e funcional

---

##  Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
